Верстаем несколько разделов сайта

В этом уроке в четырех видео шагах:

1. Мы будем верстать раздел "Ваш запрос". 
Я покажу как сделать два столбца на CSS Flexbox, 
а так же сделаем адаптивную версию сайта с помощью 
медиа запросов @media на CSS для разных устройств. 
Это достаточно простой раздел с текстом и кнопкой, 
но не менее интересный чем другие фрагменты сайта.
________
2. Мы будем верстать раздел "Услуги". 
Верстаем сетку из 3-х столбцов и 2-х рядов с помощью CSS Grid, 
а так же сделаем адаптивную версию сайта с помощью 
медиа запросов @media на CSS для разных устройств.
_________
3. Мы будем верстать раздел "Портфолио". 
Сверстаем сетку из 4-х столбцов с помощью CSS Grid,
а так же сделаем адаптивную версию сайта с 
помощью медиа запросов @media на CSS для разных устройств.
_________
4. Мы будем верстать простой раздел с текстом. 
В дальнейшем когда мы будем делать интерактивные элементы 
мы оживим этот раздел и текст в нем будет скрыт или отображен 
в зависимости от действий пользователя. 
Если нажать на заголовок текст откроется, 
при повторном нажатии скроется. 
Это может быть реализовано либо с помощью JavaScript 
или намного проще с помощью HTML тегов details и summary.
